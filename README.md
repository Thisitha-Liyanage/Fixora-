
# 📦 Stock Manager Mobile App

A modern mobile application for efficient inventory and stock management. Built with **React Native**, **Expo**, **TypeScript**, and **Firebase**, the app allows users to securely manage stock items, monitor inventory levels, and receive low-stock alerts in real time.

---

## 🚀 Features

- 🔐 Secure user authentication with Firebase Authentication
- ➕ Add new stock items
- ✏️ Update existing stock information
- 🗑️ Delete stock items
- 📋 View all inventory in a clean interface
- ⚠️ Low stock alerts
- 🔍 Search and manage stock items
- ☁️ Real-time cloud database with Firebase Firestore
- 📱 Modern and responsive mobile UI
- ✅ Form validation
- 🚀 Fast and optimized performance

---

## 🛠️ Tech Stack

| Technology | Description |
|------------|-------------|
| React Native | Mobile application development |
| Expo | React Native development platform |
| TypeScript | Type-safe JavaScript |
| Firebase Authentication | User authentication |
| Firebase Firestore | Cloud NoSQL database |
| Tailwind CSS (NativeWind) | Styling |
| Expo Router | Navigation |
